# ServiceNow CAD Exam Prep — MCQ Application

A premium, interactive Multiple Choice Question (MCQ) web application for **ServiceNow Certified Application Developer (CAD)** exam preparation.

## Features

- **200 Practice Questions** — Comprehensive coverage across all CAD exam domains
- **Strict Validation Logic** — Incorrect answers block progression until acknowledged
- **Instant Feedback** — Correct answers highlighted in green, incorrect in red
- **Score Tracking** — Real-time progress bar with correct/incorrect counters
- **Results Dashboard** — Detailed score breakdown with error review
- **Premium UI** — Glassmorphism design, smooth animations, dark theme
- **Responsive** — Works on desktop, tablet, and mobile

## Tech Stack

- **React 18** + **Vite** for fast builds
- **Vanilla CSS** with glassmorphism and micro-animations
- **Inter** font from Google Fonts
- **lucide-react** for icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```
